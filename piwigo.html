Use Piwigo 注意事项

  1. 所有的图片文件都放在 piwigo/upload/ 的目录中。

Installation

config Nginx to support

  1. /etc/nginx/nginx.conf
include /etc/nginx/conf.d/*.conf;
  1. /etc/nginx/conf.d/piwigo.conf
server {
    listen 0.0.0.0:3128;
    server_name photo;
    #root /var/www/localhost/;
    root /var/www/localhost/htdocs/piwigo;


    location ~ \.php$ {
            try_files $uri =404;
            include /etc/nginx/fastcgi.conf;
            fastcgi_pass unix:/run/php-fpm.socket;
    }
}

Install php-fm

1.  config /etc/php/fpm-php7.1/fpm.d/www.conf

  listen = /run/php-fpm.socket
  listen.owner = nginx
  listen.group = nginx
  listen.mode = 0660

2. adjust upload limited

(1) /etc/php/fpm-php7.1/php.ini

upload_tmp_dir = /tmp
upload_max_filesize = 20M

(2) /etc/ngnix/nginx.conf

client_max_body_size 20M;
client_body_buffer_size 128k;

3. add php-fpm to default service

sudo  /etc/init.d/php-fpm start
rc-update add php-fpm default

create mysql database for piwigo

mysql> create database photos;
mysql> create user 'piwigo'@'localhost' identified by 'abcdef';
mysql> grant all privileges on photos.* to 'piwigo'@'localhost';
mysql> flush privileges;

install piwigo

  1. install piwigo scripts
su 
webapp-config -h localhost -d piwigo -I piwigo 2.9.2
  1. from web browser enter http://piwgo_directory/install.php, following direction.
