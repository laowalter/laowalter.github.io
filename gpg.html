GPG setup

  • .bashrc
GPG_TTY=$(tty)
export GPG_TTY
  • import public key
gpg --import sone.bin
  • check finger print
gpg --fingerprint
  • .vimrc
Plugin 'jamessan/vim-gnupg'
  • generate keys
gpg --full-generate-key

Use GPG

  • encrypt file with recipient
gpg --output doc.gpg --encrypt --recipient blake@cyb.org doc
  • encrypt file with password
gpg -c doc
  • decrypt file
gpg --output doc --decrypt doc.gpg
