Installation (NOW)

Create the virutual env (Gentoo Linux)

cd .virutalenv
virtualenv -p /usr/bin/python3.6 blog

. ~/.virtualenv/blog/bin/activate
pip install pelican bs4 typogrify markdown pillow pycrypto pysvg-py3 piexif typogrify

Notice: pysvg-py3 is for python3, pysvg is for python2.7

Clone git

git clone ssh://git@192.168.0.199/~/blog.git 

git clone --recursive https://github.com/getpelican/pelican-plugins

Notice: the theme already in the git.

上传到github的 gh-pages

pelicanconf.py for github

一定要设置成False，否则会删除output中的.git目录

DELETE_OUTPUT_DIRECTORY = False

清空需要输出到github的目录output

rm -fr ~/blog/output
mkdir ~/blog/output
cd ~/blog/output

在output中清理干净主目录(reset to first commit)

cd ~/blog/output
git clone https://github.com/laowalter/blog.git ./
git reset HEAD
git push -f origin master

建立github指定的分支gh-pages

git branch gh-pages
git push origin gh-pages
git checkout gh-pages
git branch --set-upstream-to=origin/gh-pages gh-pages

测试上传

touch first_commit_to_gh-pages
git add first_commit_to_gh-pages
git push

每次上传新的内容

git branch gh-pages
git reset HEAD^
git push -f origin gh-pages
