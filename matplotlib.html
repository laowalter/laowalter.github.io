Matplot chinese font

  1. Problem

/home/walter/.virtualenv/python3/lib/python3.6/site-packages/matplotlib/font_manager.py:1316: UserWarning: findfont: Font family [‘sans-serif’] not found. Falling back to DejaVu Sans (prop.get_family(), self.defaultFamily[fontext]))

  1. Add Chinese font
    • Add font to system
cp simhei.ttf /usr/share/fonts/ttfs/
fc-cache -fv
    * Add following in program

```python
import matplotlib.pyplot as plt
plt.rcParams['font.sans-serif']=['SimHei']

or just change ~/.config/matplotlib/matplotlibrc

mpl.rccParams['font.sans-serif']=['SimHei']

* clear matplotlib cache

rm -fr ~/.cache/matplotlib
