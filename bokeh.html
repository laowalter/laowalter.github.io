Bokeh Server for Nginx

bokeh serve --allow-websocket-origin=192.168.0.202:8080 a_directory
  1. If add —show will start a local default browser.
  2. Above 192.168.0.202:8080 is the nginx serice port.
  3. a_directory must have a file named: main.py

Nginx

/etc/nginx/nginx.conf

server {
     listen 192.168.0.202:8080;
     server_name gentoo;
     access_log /tmp/bokeh.access.log;
     error_log /tmp/bokeh.error.log debug;

     location / {
             proxy_pass http://127.0.0.1:5006;
             proxy_set_header Upgrade $http_upgrade;
             proxy_set_header Connection "upgrade";
             proxy_http_version 1.1;
             proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
             proxy_set_header Host $host:$server_port;
             proxy_buffering off;
         }
 }
