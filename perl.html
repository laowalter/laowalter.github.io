Install perlbrew to Gentoo Linux

Install to /opt/

export PERLBREW_ROOT=/opt/perl5
curl -L https://install.perlbrew.pl | bash
perlbrew lib create base;

Install Perlbrew to Mac OSX

    \curl -L http://install.perlbrew.pl | bash
    perlbrew available
    perlbrew install perl-5.23.3
    perlbrew install-cpanm
    perlbrew lib create base;
    perlbrew switch perl-5.23.3@base

Perl mysql utf8

  1. First of all, make sure your locale is in UTF8

    • mysql “`bash /etc/my.cnf [client] default-character-set=utf8

      [mysql] default-character-set=utf8

      [mysqld] init-connect=’SET NAMES utf8’ collation-server = utf8_unicode_ci character-set-server = utf8 “`

  2. Perl script:

    perl binmode STDOUT, ":utf8"; //remove wide character warning.

  3. Connect to the database

    perl my $dbh = DBI->connect("DBI:mysql:database=nameofdb;host=localhost", "root", "password", {'RaiseError' => 1, 'mysql_enable_utf8' =>1}); // This is important to confirm utf8 connection.
