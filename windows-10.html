2.0 Windows10 active info

2.1 备份激活信息

2.1.1 备份Windows10系统激活码

  • 在运行中输入regedit命令，打开注册表编辑器，复制\HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows NT\CurrentVersion\SoftwareProtectionPlatform\ 粘贴到注册表的地址栏中；
  • BackupProductKeyDefaul的键值就是激活码

2.1.2 备份激活数据

  • Copy C:\Windows\System32\spp\store\2.0 整个目录

2.2 还原方法：

  1. 安装完成系统后。打开“服务”，找到Software Protection服务，如果该服务正在运行，请将其暂停（右键单击）。
  2. 复制备份的2.0文件夹中的cache文件夹、data.dat文件（隐藏文件）以及tokens.dat文件到C:WindowsSystem32sppstore2.0中覆盖其中的同名文件。

  3. 按Win+X快捷键，打开“命令提示符（管理员）”，输入下面的命令运行

    bash slmgr.vbs -ipk xxxxx-xxxxx-xxxxx-xxxxx-xxxxx

  4. 接着输入命令 slmgr -ato ，回车运行。如果还未显示激活，或水印还在尝试重启系统。

  1. 苹果键盘在Windows 10(64bit)的使用：AutoHotKey

; #Warn  ; Enable warnings to assist with detecting common errors.
SendMode Input  ; Recommended for new scripts due to its superior speed and reliability.
SetWorkingDir %A_ScriptDir%  ; Ensures a consistent starting directory.

; Mac keyboard to Windows Key Mappings
;=========================================

; --------------------------------------------------------------
; NOTES
; --------------------------------------------------------------
; ! = ALT
; ^ = CTRL
; + = SHIFT
; # = WIN
;
; Debug action snippet: MsgBox You pressed Control-A while Notepad is active.

#Persistent
#InstallKeybdHook
#SingleInstance force
SetTitleMatchMode 2
SendMode Input


; --------------------------------------------------------------
; Volume Control
; --------------------------------------------------------------

F10::SendInput {Volume_Mute}
F11::SendInput {Volume_Down}
F12::SendInput {Volume_Up}

; swap left command/windows key with left alt
;LWin::LAlt
;LAlt::LWin ; add a semicolon in front of this line if you want to disable the windows key

F14::SendInput {ScrollLock}
