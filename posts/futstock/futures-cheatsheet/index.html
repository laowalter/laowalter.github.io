Windows系统键盘

ALT + delete   - Del键
LWIN + i       - Ins键
LWIN + F12     - 屏幕拷贝
CTRL + ALT + F - 全屏

文化财经期货软件

Alt + o   - 显示持仓成本
ALT + r   - 隐藏K线
ALT + c   - 画线颜色切换
ALT + a   - K 线图背景颜色选白
ALT + v   - 多级别坐标联动

SHIFT + F1      - 预警设置
CTRL + R        - 反转坐标
CTRL + Alt + o  - 截图

` - 显示资金信息
