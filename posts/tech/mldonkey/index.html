安装

$ sudo emerge -avq net-p2p/mldonkey

配置

第一启动后台服务不要更改任何配置；

安装过程会自动创建一个用户“p2p”，

$ sud /etc/init.d/mldonkey start

修改配置文件

  • 首先停止服务

    /etc/init.d/mldonkey stop

  • 修改配置文件

filename: /home/p2p/mldonkey/download.ini

allowed_ips = [
  "127.0.0.1";
  "192.168.0.0/24";]

http_port = 9092 //default is 4080


filename: /etc/conf.d/mldonkey

SERVER="0.0.0.0"
PORT="9092"

  • 重启服务

    /etc/init.d/mldonkey start

防火墙添加配置打开9092管理通道

sudo iptables -A INPUT -s 192.168.0.0/24 -p tcp -m tcp --dport 9092 -j ACCEPT
