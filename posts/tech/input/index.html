filename: .bashrc

export GTK_IM_MODULE=fcitx
export QT_IM_MODULE=fcitx

If you do not enable the gtk and gtk3 USE flags, the first line should be:

export GTK_IM_MODULE=xim

If you do not enable the qt USE flag, the second line should be:

export QT_IM_MODULE=xim

说明

FCITX 4.1.2 中带的二笔是青松快版，用起来不方便。二笔论坛上提供的超强二笔的码表又有些老旧。

  1. 在 Windows 中安装极点超强二笔，从词库管理中导出码表，将前边的非数据部分删除后另存为UTF-8格式。

  2. 搜索并下载码表转换工具：jd2fcitx，将极点导出的码表转换为FCITX可以使用的格式。也就是将

    agrt 黯然 盎然
转换为
agrt 黯然
agrt 盎然
例：jd2fcitx freeime.txt > jd2fcitx.txt

  3. 然后按快版的简码设定自己编辑码表。这里需要注意，超强V键定义了很多符号，这些东西平时用的太少了。为了和下边的简码兼容，需要将v提头的编码删掉。

FCITX自带的二笔不能使用标点顶词上屏，这里需要将标点自定义一下。在码表中添加：
/ 、
, ，

; ；
最后在得到的文本文件的前边加上：
;fcitx 版本 0x03 码表文件
键码=abcdefghijklmnopqrstuvwxyz;',./
码长=4
规避字符=;iuv
拼音=@
拼音长度=4
[组词规则]
e2=p11+p12+p21+p22
e3=p11+p12+p21+p31
a4=p11+p21+p31+n11
[数据]
关于这部分的说明请看FCITX的官方网站上的WIKI。

4。使用FCITX自带的工具txt2mb将得到的文本文件转换为FCITX的数据库。

例：txt2mb jd2fcitx.txt e.mb

将mb文件复制到：~/.config/fcitx/table/

5。在FCITX中将原二笔配置的mb文件改为你自己的。

6。注消重新登录即可。 ```
