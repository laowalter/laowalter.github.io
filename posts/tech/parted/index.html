Parted 分区

  • Partition

    parted /dev/sdc
mkpart primary fat32 0% 100%
set 1 boot on

  • Format partition

    mkfs.vfat -F 32 /dev/sdc1

  1. mount iso

    sudo mkdir /mnt/windows10/
sudo mount -t auto -o -loop /path/to/windows.iso /mnt/windows10

  2. Copy all files in /mnt/windows10 to usbstick

    mount /dev/sdc1 /mnt/usb
screen -S cp
cd /mnt/usb/
sudo cp -fr /mnt/windows10/* .
