使用xelatex

模板一

filename: 模板一(简体)
\documentclass[11pt,a4paper]{article}
\usepackage{fontspec,xltxtra,xunicode}
\usepackage[slantfont,boldfont]{xeCJK}

\setCJKmainfont[BoldFont=Adobe Heiti Std,ItalicFont=Adobe Kaiti Std]{Adobe Song Std}
\setCJKsansfont{Adobe Heiti Std}
\setCJKmonofont{Adobe Song Std}

\setCJKfamilyfont{song}{Adobe Song Std}
\setCJKfamilyfont{hei}{Adobe Heiti Std}
\setCJKfamilyfont{kai}{Adobe Kaiti Std}

\newcommand{\hei}{\CJKfamily{hei}}
\newcommand{\song}{\CJKfamily{song}}
\newcommand{\kai}{\CJKfamily{kai}}

\title{\LaTeX中文博客模板}
\author{wangl393}
\date{\today}

\begin{document}
{默认效果 宋体}\\
 {\bfseries 粗体效果 设置成黑体}\\
 {\itshape 斜体效果 设置成楷体}\\
\kai  CJKfamily为楷体效果\\
\hei  CJKfamily为黑体效果\\
\end{document}

模板二(繁体)

filename: 模板二
\documentclass[11pt,a4paper]{article}
\usepackage{fontspec,xltxtra,xunicode}
\usepackage[slantfont,boldfont]{xeCJK}

\setCJKmainfont[BoldFont=Adobe Heiti Std,ItalicFont=Adobe Kaiti Std]{Adobe Ming Std}
\setCJKsansfont{Adobe Heiti Std}
\setCJKmonofont{Adobe Song Std}

\setCJKfamilyfont{song}{Adobe Ming Std}
\setCJKfamilyfont{hei}{Adobe Fan Heiti Std}
\setCJKfamilyfont{kai}{Adobe Kaiti Std}

\newcommand{\hei}{\CJKfamily{hei}}
\newcommand{\song}{\CJKfamily{song}}
\newcommand{\kai}{\CJKfamily{kai}}

\title{\LaTeX中文博客模板}
\author{wangl393}
\date{\today}

\begin{document}
{默认效果 宋体}\\
{繁体修心沒有終點}

 {\bfseries 粗体效果 设置成黑体}\\
 {\itshape 斜体效果 设置成楷体}\\
\kai  CJKfamily为楷体效果\\
\hei  CJKfamily为黑体效果\\
\end{document}

安装 xetex、xelatex、latex

emerge dev-texlive/texlive-xetex
emerge dev-texlive/texlive-langcjk
emerge dev-tex/cjk-latex

安装其他中文宏包

cd /usr/share/texmf-dist/tex/generic/
git clone https://github.com/CTeX-org/ctex-kit.git
texhash

安装中文字体

  1. Copy fonts (Adobe Simsun) to /usr/share/fonts/

  2. Then running:

    fc-cache -f
fc-list

Vim plugin for latex (lervag/vimtex)

Install

Install plugin

filename: .vimrc
Plugin 'lervag/vimtex'

let g:vimtex_compiler_latexmk = {
    \ 'options' : [
    \   '-pdf',
    \   '-pdflatex="xelatex --shell-escape %O %S"',
    \   '-verbose',
    \   '-file-line-error',
    \   '-synctex=1',
    \   '-interaction=nonstopmode',
    \ ]
    \}

then save and quit then restart vim enter :BundleInstall

Install required package

emerge -avq dev-tex/latexmk
