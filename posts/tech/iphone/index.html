Sat Feb 17, 2018

Home iphone

    • Connect to linux
    • Backup whole iphone
    • Mount / umount

Connect to linux

sudo emerge -avq app-pda/ifuse
sudo emerge -avq app-pda/usbmuxd
suod usbmuxd -f -v
sudo idevicepair pair

Backup whole iphone

idevicebackup -i backup --full backupdir

Mount / umount

  • setup /etc/fuse.conf

ifuse 定义的目录就是 /media/iPhone, 这是一个固定名称目录

user_allow_other

  • mkdir /media/iPhone

    make /media/iPhone
chmod 777 /media/iPhone/

  • Mount iphone

    ifuse /media/iPhone --root

  • Umount iphone

普通用户即可

fusermount -u /media/iPhone
