Install

Gentoo

    $ sudo layman -a foxiverlay
    $ sudo eix-update
    $ emerge -avq net-p2p/go-ethereum
    $ emerge -avq net-p2p/ethminer

Create get account

   $ geth account new

Create ethereum wallet

  1. Go to https://github.com/ethereum/mist/releases

  2. Download windows version, tutotialhttps://medium.com/@attores/step-by-step-guide-getting-started-with-ethereum

  3. etherminerhttps://github.com/ethereum-mining/ethminer/releases

Mining

Use Internet Server

  • ethermine.org server list

  • Command line

    ethminer.exe -G -P stratum+ssl://0x3DC7F73Ad59e55f41fc8a40F0304F9e243358f9a.miner1@asia1.ethermine.org:5555

With geth

CPU Mining

  • linux

    geth --mine --minerthreads=4 --cache 4096
GPU Mining

With ethermine

https://github.com/ethereum-mining/ethminer/releases
  • windows

Note: 0.14rc3版本 -F 开关说是过时了，用-P替代，但是用-P程序就崩溃了。所以只能用-F

ethminer.exe -G -F http://192.168.0.199:8545/?miner=0x3DC7F73Ad59e55f41fc8a40F0304F9e243358f9a

Multi Machine Mini

Start as a server

ethminer --rpc --rpcaddr 192.168.0.199 --rpcport 8545
