Add user to linux

useradd -m -G users,wheel username
passwd username

Add user to Samba

smbpasswd -a user

Add a user to a group

usermod -a -G examplegroup exampleusername

Convert a user to nologin shell

usermod -s /sbin/nologin exampleusername
