Xorg  最小安装

  1. 检测显卡：
#dmesg | grep video
#lspci  | grep -i VGA
  1. 查看电脑的显卡，为装显卡驱动做准备 
 设置 /etc/portage/make.conf 文件中两个重要的变量：

（对鼠标和键盘的支持）
INPUT_DEVICE="evdev"

（对 nVidia 显卡）
VIDEO_CARDS="nvidia"

（或，对 ATI Radeon 显卡）
VIDEO_CARDS="fglrx"

（或，对 VMware 虚拟机显卡）
VIDEO_CARDS="vmware"
  1. 安装 xorg-server
# emerge -pv xorg-server
# echo "x11-base/xorg-server udev" >> /etc/portage/package.use
# emerge xorg-server
  1. 初始化环境变量 
# env-update
# source /etc/profile
  1. 使用 startx： 测试 X，安装 twm xterm
# emerge twm
# emerge xterm
# startx

测试成功，卸载 twm xterm

emerge --unmerge twm xterm

注意事项 现在比较新的版本的 Xorg 使用 evdev 替换了 keyboard 和 mouse ，确保 udev 标记在安装 xorg-server 时被启用：# echo “x11-base/xorg-server udev” >> /etc/portage/package.use
